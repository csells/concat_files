# Usage

```shell
Usage: dart concat.dart <folder>
Example: dart concat.dart /path/to/your/top/level/folder
```
# Output Example

File "/source/com/foo/bar.java":

```java
bar.java content
```

File "/source/com/foo/bar.xml":

```xml
bar.xml contents

```

File "/foo.java":

```java
foo.java content
```

File "/foo.xml":

```xml
foo.xml content
```
